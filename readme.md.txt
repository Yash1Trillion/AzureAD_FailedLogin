# Azure AD Failed Login Analysis with MITRE ATT&CK Mapping

## Overview
Detect failed login attempts in Azure AD, visualize alerts, and map them to MITRE ATT&CK techniques. This simulates a SOC workflow.

## Tech Stack
- Azure AD, Azure Sentinel
- Python 3.x, Pandas, Plotly, Kaleido

## Features
- Failed login detection
- Analysis by user & location
- MITRE ATT&CK mapping
